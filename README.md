# Canonical Cross-Reference Graph

A general-purpose graph dataset and visualization project for exploring canonical Bible cross-references as a relational network.

----

A Gephi-centered graph project for visualizing biblical cross-reference data.

This repository currently contains a master `.gephi` file built from the OpenBible.info cross-reference dataset. The goal is to provide a workable network graph that can be opened, explored, filtered, styled, and exported through Gephi.

Main file:

[cross_references_canonical.gephi](gephi/cross_references_canonical.gephi)

## Current Focus

* maintain a master Gephi project file
* visualize biblical cross-references as a node-and-edge graph
* support layout, filtering, and graph analysis work in Gephi
* provide a foundation for future Scrollmapper or other graph-based tools

## Source

Cross-reference data is based on the OpenBible.info Bible Cross References dataset.

## Status

Early setup. More data exports, scripts, notes, and documentation may be added later.
